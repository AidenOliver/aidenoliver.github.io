Ristorante Con Fusion

Responsive website made with Bootstrap, HTML, CSS(Less, Scss) and JavaScript(JQuery)

CAPStone Project

Ristorante Con Fusion is a website for a fictional restaurant that serves Italian and Indian food. The site contains carousel sliders that show images every 5 seconds and then rotates another image. The carousel has buttons so the user can pause or go back to the previous image.

Mobile-first

When I first started building the website I started from mobile-up which meant starting the layout based on phone-size and then added to it so it could fit with desktop, laptop, ipad, etc.. The navigation collapses into a button that will display the navigation vertically when the screen width drops below a certain width. This makes it easier for mobile users to tap the navigation bar with their phone.

Responsive development plays an important role in websites and apps so it's important to make sure every aspect of the site should be smooth for every device.

I used Bootstrap when working on the project and it has shown great results. Taking advantage of the predefined classes and attributes yielded a faster time to get the website done while making sure it was fully responsive at the same time.
